U5L2 If Statements

  • conditional--a statement that has a condition or requirement (ex: If a person is 18, then that person can vote)
  • Writing conditionals in JS:
    • if (condition is true){
      run code in the curly brackets
      }
  • If the condition is false, the code inside the corresponding brackets will not run.

U5L3 Inequalities in JS

  • Use >= for greater than or equal to (Note: this means greater than OR equal to, not AND)
  • Use <= for less than or equal to
  • You can NOT use = in if statements b/c = means "assign" in coding
  • Use == to check if two numbers are equal. You can also use == to check if two strings are equal to each other (Note: capitalization and spacing must be same for statement to be considered true)
  • In JS, numbers can be considered equal to strings if the number matches the text of the string (number is converted to string). Unlike many other languages, JS will try to match the two values to the best of its ability.
    • Ex: "18" == 18 is considered a true statement in JS
    • let age = "16"; age == 16 is true
    • let age = 17; age == "17" is true
  • Use === to check if two statements are EXACTLY equal--in both characters and data type
    • Ex: 18 === "18" is considered a false statement
